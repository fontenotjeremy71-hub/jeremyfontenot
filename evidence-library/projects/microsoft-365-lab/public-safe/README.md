This directory contains public-safe aggregate summaries derived from private raw exports captured on 2026-06-05 in a personal Microsoft 365 tenant.

Reviewed seven sanitized lab identity records, eight group records, directory-role assignments, sign-in visibility, and directory-audit visibility. Names, UPNs, IP addresses, event IDs, object IDs, tenant identifiers, application identifiers, device identifiers, precise locations, and privileged identities are withheld.

## Scope

- Personal tenant only.
- Captured-state evidence, not live telemetry.
- Aggregate review evidence does not prove employer-system administration, enterprise scale, production ownership, recurring assurance, or current subscription status.
- Global Administrator assignments reviewed: 3. Named identities: redacted.
- Consent records reviewed: 16. Permission categories: Directory, Sites, Files, Devices, Audit, Applications. Raw client, resource, principal, and tenant identifiers are withheld.
- Sign-in review covers the exported 30-day window ending 2026-06-05. Only aggregate record counts, client/application categories, interactivity, and Conditional Access result categories are published.

The private source exports must remain outside the public publication surface.
